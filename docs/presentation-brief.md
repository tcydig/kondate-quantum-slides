# 量子献立提案アプリ プレゼンテーション概要

## 基本情報

| 項目 | 内容 |
|------|------|
| タイトル | 量子献立提案アプリ（仮） |
| 発表時間 | 5分 |
| 対象者 | 投資家・エンジニア |

## コアメッセージ

**三方よしの精神で、誰も取り残さない最適化を実現する。**

## 背景・問題意識

### 既存の取り組みとその限界

- 量子アニーリングを用いた献立提案の先行事例はすでに存在する
- しかし、いずれも **栄養価やコスト** のみを最適化対象としている
- **実際に献立を調理する人（調理員）の負荷** が考慮されていない

### 構造的な課題

- 古典コンピューターの技術的限界と、工業化による高度な分業化が進んだ結果、技術リテラシーの格差が生まれた
- 技術リテラシーが低い領域（例：学校給食の調理現場）は効率化が進まず、依存先である効率化が進んだ領域からの要求に **人的リソースで対応** せざるを得ない状況にある
- この構造的な不均衡こそが、本アプリの問題意識の根幹である

## アプリ概要

- **技術**: 量子アニーリング
- **用途**: 学校給食の献立提案
- **特徴**: 栄養価・コストに加え、**調理員の負荷** を最適化パラメータに含めている
- **差別化**: 三方よし（栄養価・コスト・調理負荷）の同時最適化

## ビジネスプラン

### 事業形態

- **NPO法人** として設立予定（詳細は今後検討）
- 自治体・教育機関との契約による運営を想定

### 最適化パラメータの拡張：環境負荷

- **環境負荷**（フードマイレージ、食品ロスなど）も最適化パラメータに加えたい
- 今回の発表に間に合えば盛り込む。間に合わなければ発表後に追加する
- 三方よし → 四方よし（栄養価・コスト・調理負荷・環境）への拡張

### スケール戦略

```
Phase 1: 学校給食（現在）
    ↓
Phase 2: 病院給食・社会福祉施設
    ↓
Phase 3: 家庭内の献立提案（朝食・夕食）
```

- 子どもの食事は昼（給食）だけでなく、朝・夜もある。家庭内の献立提案まで視野に入れることで、子どもの食の最適化を一日を通して実現できる

### 食育エコシステム構想

献立提案を起点に、GIGAスクールのタブレット環境や保護者連携を活かした食育エコシステムへの展開も視野に入れている。（※量子アニーリングとは直接関係しないが、スケールの方向性として）

- **学校（タブレット活用）**
  - 給食の内容に関する教育動画をAIで自動生成
  - 食材や栄養に関するクイズ・問題の自動作成
  - 地産地消の食材を使用している場合、生産農家の紹介動画を配信

- **保護者連携アプリ**
  - 子どもの給食内容を保護者にリアルタイム連携
  - 給食で子どもが食べられた野菜がある場合、地元農家から直接購入できる導線を提示
  - 給食 → 家庭の食卓へつなげる体験の一貫性

- **目指す姿**: 献立最適化アプリを核として、学校・家庭・地域の農家をつなぐ食育プラットフォームへ発展

## 現場ヒアリング結果

調理員への聞き取り調査から、以下の課題が明らかになった。

### 調理員が抱える不満・課題

- **早朝出勤・前日準備の負担**: 調理に時間がかかるメニューが多いと、早朝に出社したり前日から仕込みを行う必要がある
- **献立変更が通らない**: 調理時間が足りないと感じ、事前に栄養士へ献立の見直しを依頼しても「もう決まっているから変えられない」と断られる
- **立場の格差**: 栄養士は市役所勤務（公務員）であるのに対し、調理員は立場が弱く、意見が反映されにくい構造がある
- **人手不足の悪循環**: こうした負担や不満から離職者が増え、さらに人手不足が深刻化している

### ヒアリングから見える本質的な問題

現在の献立作成プロセスでは、**調理員の声が構造的に反映されない**。栄養価やコストは数値化・最適化の対象となっているが、調理現場の負荷は「見えないコスト」として放置されてきた。本アプリはこの構造的な不均衡を、量子アニーリングによる多目的最適化で解消することを目指している。
